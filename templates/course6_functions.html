Урок 1: Введение в функции

Что такое функция?

Функция — это именованный блок кода, который можно вызывать многократно. Это как рецепт: вы записываете его один раз, а потом можете готовить по нему много раз.

# Объявление функции
def приветствие():
    print("Привет!")
    print("Добро пожаловать в мир Python!")

# Вызов функции
приветствие()  # Выведет два принта

Урок 2: Аргументы и параметры

Параметры функций

Функции могут принимать входные данные — аргументы. Это позволяет делать функции гибкими.

# Функция с параметрами
def приветствие_по_имени(имя):
    print(f"Привет, {имя}!")

def сложение(a, b):
    результат = a + b
    print(f"Сумма: {результат}")

# Вызовы
приветствие_по_имени("Анна")  # Привет, Анна!
сложение(5, 3)                  # Сумма: 8
