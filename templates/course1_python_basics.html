Урок 1: Что такое Python?

Что такое Python?

Python — это «язык», на котором мы отдаем команды компьютеру. Python выбран не случайно: он похож на обычный английский, поэтому его легко читать и писать.

Представьте, что вы учите малыша разговаривать: сначала нужно показать ему буквы, потом научить складывать их в слоги, а затем — в слова. Так и с Python — мы начнем с самых основ.

Установка и настройка

Как «познакомить» ваш компьютер с Python и в какой программе (среде разработки) писать код, чтобы было удобно. Это как настроить рабочее место перед тем, как лепить скульптуру.

# Первая программа на Python
print("Привет, мир!")
print("Я начинаю изучать Python")

Урок 2: Переменные и типы данных

Переменные

Переменная — это коробочка с именем, в которую можно положить какие-то данные. Удобно, чтобы не держать всё в голове.

имя_ученика = "Иван"
возраст = 25
print("Привет, меня зовут", имя_ученика)
print("Мне", возраст, "лет")

Типы данных

  • Числа (int, float): Целые (5, 100, -3) и дробные (3.14, 2.5).
  • Строки (str): Текст. Всегда пишется в кавычках. "Привет", 'Как дела?'.
  • Булевы значения (bool): Правда или ложь. True (Истина) или False (Ложь).

Урок 4: Ввод и вывод данных

Вывод данных

print() — команда, которая показывает текст на экране (выводит данные).

print("Привет, мир!")
print(5 + 3)
print("Результат:", 42)

Ввод данных

input() — команда, которая ждет, пока вы что-то напишете с клавиатуры, и запоминает это (вводит данные).

имя = input("Как тебя зовут? ")
print("Приятно познакомиться,", имя)

возраст = input("Сколько тебе лет? ")
print("Через год тебе будет", int(возраст) + 1)
